# 薪资管理系统简介 python854

## 项目概述

薪资管理系统是基于 Python 语言开发的一套企业薪资管理解决方案。系统集成了 HTML 用于前端展示，并与 MySQL 数据库配合，实现数据存储和管理功能。

## 技术栈

- 后端开发语言：Python
- 前端开发技术：HTML
- 数据库管理系统：MySQL

## 系统运行环境

- 数据库版本：MySQL 8
- Python 版本：3.7.7
- 操作系统兼容性：Windows 10/MacOs

## 开发工具

- 集成开发环境：PyCharm
- 数据库可视化工具：Navicat

## 功能模块

系统主要包含以下功能模块：

- 薪资数据录入
- 员工信息管理
- 薪资计算与发放
- 报表统计与分析

## 目录结构

具体的目录结构如下：

```
project/

├── app/                 # 应用目录

│   └── utils/           # 工具函数
│

│
└── static/              # 静态资源
```
